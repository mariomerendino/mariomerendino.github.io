Base Changer
During my computer organization course, I needed a tool to convert numbers from Binary to Hex to Octal and to decimal. There are plenty of resources out there however, few of them let you use signed binary numbers, and if they do let you use signed numbers, they do not let you change which bit is the signed one. This tool allows you to do all of that very easily! I created this project in Xamarin.Forms using C# for cross platform mobile development.
[iOS and Android]
